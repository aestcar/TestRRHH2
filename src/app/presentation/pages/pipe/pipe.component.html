Página de prueba de la pipe skillsLevels

Este es un pipe custom que en base a un número entre 1 y 5 muestra un emoji con un texto

Normal Mode

    @for (level of skillLevels(); track level) {
  • {{ level }}: {{ level | softSkillLevel }}
    • }

Compact Mode

    @for (level of skillLevels(); track level) {
  • {{ level }}: {{ level | softSkillLevel: 'compact' }}
    • }
